# keyboards
A comparison and review of mechanical keyboards

| Joins? | Yes (not solid) | No | Yes (magnet) | No |

| Learning | Easy | Hard | Med | Hard | 
| Ortho | No | Yes | No | Yes |

| Removable? | Yes | Yes | Yes | No |
